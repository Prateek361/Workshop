📘 About This Project

The AI Waste Segregation System is a web-based application that integrates Google Teachable Machine with TensorFlow.js to classify waste in real-time using computer vision techniques.

The model is trained on multiple waste categories and deployed directly in the browser, eliminating the need for a backend server.

This project demonstrates the practical application of Machine Learning in environmental sustainability and smart waste management.
